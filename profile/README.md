---

## 🚀 What is Pit-1?

Pit-1 is a developer-first **AI infrastructure agent** that runs in your terminal.

Describe what you want — or just pass your code —  
Pit-1 will design and generate production-ready infrastructure.

---

## ✨ What you can do

- 🧠 **Natural language → Infrastructure**
  ```bash
  pit1 "node api + redis + private vpc"
  ```

- 💻 **Code → Infra recommendation**
  ```bash
  pit1 from-code ./myapp
  ```

- 🧩 **Diagram → IaC + CLI (coming soon)**

- 🔁 **IaC → Diagram (coming soon)**

---

## ⚙️ CLI-first Experience

No dashboards. No YAML hell.

```bash
# generate infra
pit1 "fastapi + postgres + aws"

# preview architecture
pit1 plan

# deploy
pit1 apply
```

> Run your code. We’ll build the cloud.

---

## 🧠 How it works

```
Input (Code | Text | Diagram | IaC)
                ↓
        AI Strategy Engine
                ↓
   Infrastructure (IaC + CLI + Diagram)
```

Pit-1 uses an AI-driven **infra strategy layer** to translate intent into real systems.
